# .dotfiles 

Archivos de configuración tanto para WSL2 y Arch Linux

Contenidos:  
1. Kitty        - Emulador de terminal para Arch Linux
2. Nvim         - Editor de código basado en la terminal de comandos. Vim pero con más capacidades de personalización.
3. TMUX         - Multiplexor de terminal. Mejora el workflow teniendo múltiples ventanas en una sola terminal.
4. ZSH          - Interprete de línea de comando basado en Bash que ofrece la capacidad mayor personalización permitiendo usar plugins.
5. Powerline10K - Theme para ZSH
